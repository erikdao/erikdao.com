## Project Overview

Personal blog and portfolio site (erikdao.com) built with **Astro v6**, **Tailwind CSS v4**, and **TypeScript**. Content is authored in Markdown with YAML frontmatter using Astro Content Collections.

## Commands

- **Dev server**: `yarn dev` — runs Astro dev server with HMR
- **Production build**: `yarn build` — builds to `dist/` (static output)
- **Preview**: `yarn preview` — preview the production build locally

No test runner or linter is configured.

## Architecture

**Astro config**: `astro.config.mjs` — defines integrations (sitemap), markdown plugins (remark-math, rehype-katex), and Vite plugins (@tailwindcss/vite).

**Content collections** (defined in `src/content.config.ts`):
- `src/content/dev/` → `dev` collection
- `src/content/machine-learning/` → `machine-learning` collection
- `src/content/life/` → `life` collection
- `src/content/violin/` → `violin` collection
- Featured posts: queried at render time by filtering `featured: true` across all collections

**Layouts**: Astro components in `src/layouts/`. `Base.astro` is the base layout; `Post.astro` wraps posts with article header and Disqus comments.

**Components**: Astro components in `src/components/` — `SiteHeader.astro`, `SiteFooter.astro`, `MetaInfo.astro`, `PostList.astro`, `FeaturedPosts.astro`, `FeedHeader.astro`, `PageHeader.astro`.

**Pages**: File-based routing in `src/pages/`. Category pages use `[slug].astro` with `getStaticPaths()` for dynamic routes.

**Data files**: `src/data/` — `site.json` (metadata), `navigation.json` (nav items), `books.json`, `todo100.json`.

**Static assets**: `public/` — images, resources, favicon. Served at root path.

**Styling**: Tailwind v4 CSS-based config in `src/styles/global.css` (uses `@import "tailwindcss"` and `@theme` directives). Imported in `Base.astro`.

## Post Frontmatter

```yaml
---
title: 'Post Title'
date: 'YYYY-MM-DD'
tags: ['category']
featured: true          # optional, includes in featured posts
summary: "Description"
socialImage: '/images/path.png'
---
```

## Key Features

- **LaTeX math**: `$$equation$$` syntax rendered by remark-math + rehype-katex
- **Syntax highlighting**: Shiki (built-in, zero client JS) with `github-light` theme
- **HTML/CSS minification**: Handled automatically by Astro/Vite in production
- **Disqus comments**: Embedded in Post layout
- **Sitemap**: Auto-generated by @astrojs/sitemap
- **Google Analytics**: Conditionally loaded in production via `import.meta.env.PROD`

## Adding a New Post

1. Create a `.md` file in the appropriate category directory under `src/content/`
2. Add required frontmatter (title, date, tags, summary)
3. Place images in `public/images/<category>/`

## Adding a New Category

1. Create directory `src/content/<category>/`
2. Add collection in `src/content.config.ts` with a `glob` loader
3. Create `src/pages/<category>/index.astro` (category listing)
4. Create `src/pages/<category>/[slug].astro` (post pages with `getStaticPaths`)
5. Add entry to `src/data/navigation.json`
